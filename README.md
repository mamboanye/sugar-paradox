# The Sugar Paradox

Countries that consume more sugar are fatter. Across 37 Sub-Saharan African countries, national sugar supply predicts women's obesity at r = 0.50 after controlling for income, urbanization, and vegetable oil supply. Drop any single country and the result holds.

Follow the same countries over time and the association vanishes. Six within-country specifications converge on null. The reason: entity and year fixed effects absorb 99.4% of obesity variance, leaving 0.6% for food supply to explain. The null extends to all 10 FAOSTAT food groups. A soybean oil case study shows that standard robustness tests (clustered SEs, leave-one-out, Oster bounds) pass for a variable that is pure co-trending. Globally, 98% of 200 countries show linear obesity trends with R2 >= 0.90.

The only predictor of which countries gained more obesity is how much they had to start with.

This repository contains the data, code, figures, and manuscript.

## Reproduce

```
git clone https://github.com/mamboanye/sugar-paradox.git
cd sugar-paradox
python run_all.py
```

Requires [uv](https://docs.astral.sh/uv/). Dependencies install automatically. Works on macOS, Linux, and Windows.

Every number in the paper traces to one of 11 scripts in `scripts/`. The pipeline runs in about two minutes and produces all derived data, figures, and tables from the raw sources included in the repo.

## Data

All source data ship with the repo in `data/raw/`. No external downloads required.

| File | Source |
|------|--------|
| `FoodBalanceSheets_E_Africa.zip` | [FAOSTAT](https://www.fao.org/faostat/en/#data/FBS) |
| `NCD_RisC_Lancet_2024_BMI_age_standardised_country.csv` | [NCD-RisC](https://www.ncdrisc.org/) |
| `NCD_RisC_Lancet_2024_Diabetes_age_standardised_countries.csv` | [NCD-RisC](https://www.ncdrisc.org/) |
| `wdi_ssa_food_health_panel_2000_2024.csv` | [World Bank WDI](https://databank.worldbank.org/) |
| `ssa_faostat_ncdrisc_analysis_panel_2010_2022.csv` | Merged 37-country panel (962 rows) |

## Structure

```
scripts/          11 numbered Python scripts, run sequentially
data/raw/         source data (included)
data/derived/     intermediate outputs (generated by pipeline)
figures/          6 publication figures (generated)
tables/           4 source tables (generated)
paper/latex/      manuscript source and compiled PDF
```

## Manuscript

The compiled paper is at `paper/latex/main.pdf`. To rebuild from LaTeX source: `cd paper/latex && bash build.sh`.
