# The Sugar Paradox

Replication package for "The Sugar Paradox: National food supply predicts obesity across countries but not within them."

By [Mambo-Mary Nzong Anye](https://orcid.org/0009-0000-1452-003X) (Mississippi State University) and [Bryan Anye Nsoh](https://orcid.org/0009-0003-1495-6281) (University of Tennessee, Knoxville).

## Paper

- **Preprint (named):** `paper/preprint/main.pdf`
- **Blinded (conference):** `paper/preprint/main_blinded.pdf`
- **Original format:** `paper/latex/main.pdf`

## Reproduce

```
git clone https://github.com/mamboanye/sugar-paradox.git
cd sugar-paradox
python run_all.py
```

Requires [uv](https://docs.astral.sh/uv/). Dependencies install automatically. Works on macOS, Linux, and Windows. Runs in about two minutes.

Every number in the paper traces to one of 15 pipeline scripts in `scripts/`. All source data are included in `data/raw/`. No external downloads required.

## Structure

```
scripts/              15 pipeline Python scripts, run sequentially
data/raw/             source data (FAOSTAT, NCD-RisC, WDI)
data/derived/         intermediate outputs (generated by pipeline)
figures/              publication figures (generated)
tables/               source tables (generated)
paper/preprint/       Michaillat-template preprint (named + blinded)
paper/latex/          original manuscript source and compiled PDF
run_all.py            cross-platform pipeline runner
```

| 03b_food_groups_twfe.py | Full cascade for all 10 FAOSTAT food groups |
| 04_trend_decomposition.py | Trend R2, FE absorption, SE inflation |

| 10_tables.py | 4 tables (descriptives, cascade, robustness, change) |

## Data Sources

- FAOSTAT Food Balance Sheets (Africa), downloaded 2024
- NCD-RisC Lancet 2024: BMI and diabetes age-standardised country estimates
- World Development Indicators (pre-downloaded panel)

## Key Results

All standard errors are clustered at the country level. The analysis panel contains 37 SSA countries, 2010-2022, 962 rows (country x year x sex).

- Cross-sectional sugar-obesity partial r = 0.50 (p = 0.002)
- Within-country TWFE sugar t = -0.64 (null under six specifications)
- Entity + year FE absorb 99.4% of obesity variance
- 0/10 FAOSTAT food groups survive TWFE or first differences
- Total energy, fat, and protein supply are also null under TWFE and first differences; the lone detrended total-energy signal weakens with year effects and has both lag and lead signal
- Soybean oil passes standard tests but lead-lag ratio = 0.99 (pure co-trending)
- 98% of 200 countries globally show linear obesity trends (R2 >= 0.90)
- GDP reverses sign after detrending (t = -4.12), confirming method has power
- Only predictor of obesity change is initial obesity level (t = 3.45)
